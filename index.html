MPC-Chord-Finder

  1. This is a tool to show you how to help you locate chords on your AKAI MPC

    MPCs are drum samplers that look like this.

    Select a single note sample (poly mode) and tune it so that PAD1 is C.
    Hit 16/12LEVELS (tuning).
    Grab your real book and get ready.

    Next

  2. Select 12 or 16 pads mode
    (you can change that anytime)

    Next

  3. Select a key.
    If your chord is called E minor, this would be "E"

    Next

  4. Select a chord type.
    ("minor", "major", etc.)

    Next

  5. And see the chord displayed here.

    Next

  6. Add the chord to your chord set...

    Next

  7. ...et voilà.

    Next

  8. Store the URL for your set or share it with the world!

    Next

  9. ...or print your set in landscape mode and take it to the stage

    Next

  10. You can also clear all chords from a set using this button.
    Have fun!
